先安装 api server: https://github.com/lincenying/mmf-blog-api

```
// 安装cooking
npm install cooking-cli -g

// 安装依赖
npm install  or  yarn

// 生产环境
npm run build

// 开发环境
npm run dev
```

首页
http://localhost:8080
